

# Validated PDC Exam for Early Computing Education & Research

This is a public hosting site for a validated instrument (multiple choice
exam) for parallel and distributed computing (PDC) for early computing
curricula (CS0/1/2).  

## Version

* Version 0.1.1 - minor fixes
* Version 0.1.0 - initial public version

## Formats

* JSON
* markdown
* Moodle module
* Canvas module

## Parsers

Several parsers have been provided (`questions.py`) to assist in translating
the exam to other formats.
